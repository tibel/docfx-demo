## Diagrams
Mermaid can be used to create beautiful diagrams.

``` mermaid
graph LR
    A -- text --> B -- text2 --> C
```

``` mermaid
sequenceDiagram
    Alice->John: Hello John, how are you?
    Note over Alice,John: A typical interaction
```
